Projects

Scheme Interpreter

An interpreter for a subset of the Scheme language that I developed in the course CS 61A: Structure and Interpretation of Computer Programs, implemented with Scheme macros and tail recursion. The basis of this interpreter is the REPL loop (i.e. Read-Print-Eval-Loop). The interpreter can be run in the terminal and is capable of parsing and executing Scheme code.

Due to CS61A course policy, the Github repository containing this project has been made private.

Ants Vs SomeBees

A tower defense game where the player populates their colony with a variety of unique bees that perform various different kinds of actions. I developed this project as part of the course CS 61A: Structure and Interpretation of Computer Programs. The project is programmed in Python using its object-oriented system.
